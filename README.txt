pyrow
=====

A Python application for recording training or race sessions from Concept2 PM3 or PM4 devices onto a computer.

This project draws inspiration from other python ergometer programs.
Sam Gambrell's Pyrow program is available at http://www.newhavenrowingclub.org/pyrow/
Vishal Patel's RowMetrics program is available at http://www.vishalpatel.net/projects/rowmetrics.php
